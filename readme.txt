== Description ==

Mp3 Quran Translations in 52 languages and 54 different reciters plugin is the first WordPress plugin that allows you to display a playlist for the mp3 files in English translations to understand the meaning of the Quran in english, powered by TrueMuslims.Net.

== Installation ==

* Upload the Mp3 Quran Translations plugin to wordpress site or blog, Activate it.
* Go to Mp3 Quran from menu.
* Copy yellow code like this: [mp3Quran]*[/mp3Quran] and past code in your post or page.
* Done!
